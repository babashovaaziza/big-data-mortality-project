# Big Data Mortality Analysis using PySpark

This project analyzes mortality and suicide trends using PySpark on Amazon EMR.  
The analysis includes:
- Suicide-to-total mortality ratio over time
- Gender distribution of suicide incidents
- Male-to-female suicide ratio trends

Technologies used:
- PySpark
- Hadoop HDFS
- Amazon EMR
- JupyterHub
- Pandas
- Matplotlib
